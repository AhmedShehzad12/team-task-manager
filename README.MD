Project Name: Team Task Manager

Introduction:
This project is a web application designed to solve real-world problems in a simple and
efficient way. It allows users to interact with the system easily and perform tasks online
without any difficulty.

Project Overview:
Team Task Manager is a web-based platform where users can:
Create and manage tasks
Assign tasks to team members
Track task progress
Organize workflow efficiently
It provides separate modules for users and admins to ensure smooth task handling and
management.

Problem Statement:
Many users face difficulties in managing tasks / bookings / data manually. This project
aims to provide a digital solution that saves time, reduces errors, and improves user
experience.

Solution:
This application provides an easy-to-use interface where users can perform all required
actions online. It automates the process and makes everything faster and more efficient.

Tech Stack:

Frontend:
HTML
CSS
JavaScript
React.js

Backend:
Node.js
Express.js

Database:
MongoDB
Other Tools:

Git & GitHub
REST APIs

Features:

User Authentication (Login/Register)
Create, Update, Delete Tasks
Assign Tasks to Team Members
Track Task Status (Pending / Completed)
Admin Panel for Management
Search & Filter Tasks
Responsive UI

Installation & Setup:

1.Clone the repository
git clone https://github.com/your-username/team-taskmanager.git
cd team-task-manager

2.Install dependencies
npm install

3. Run Backend
cd Backend
npm start

4.Run Frontend
cd Frontend
npm start

Environment Variables:

MONGO_URI=your_mongodb_connection
PORT=5000
JWT_SECRET=your_secret_key
